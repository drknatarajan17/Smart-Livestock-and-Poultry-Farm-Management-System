# 🐔 Smart Livestock & Poultry Farm Management System

A simple, user-friendly Farm Management System developed using **Python**, **Streamlit**, and **Pandas** to help small and medium farmers manage their daily farm operations efficiently.

The application supports **Native Chicken (Nattu Kozhi), Goat, and Cow farming** by maintaining livestock records, expenses, income, sales, medical history, employee details, and profit analysis through an easy-to-use dashboard.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Smart-Livestock-and-Poultry-Farm-Management-System.git
```

Go to the project folder

```bash
cd Smart-Livestock-and-Poultry-Farm-Management-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
